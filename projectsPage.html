Bachelors thesis project

My bachelors thesis program's goal was to get information about Github projects and evaluate these informations. The goal of my thesis was to replicate a study about the effectivity of visual issue reports on GitHub. To evaluate the effectivity of visual issue reports, I made a program in C# which collected and evaluated information about Github projects. Firstly my program searched gitHub repositories which corresponded to the filters I used and then it collects the Issues. After the collection of informations my program evaluated the collected informations and saved the informations and evaluation results into text files.
